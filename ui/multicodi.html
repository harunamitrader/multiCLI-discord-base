エージェント設定ガイド

エージェントを追加するには .env ファイルに以下を記載してサーバーを再起動してください: 

# Claude エージェント "hanako"
AGENT_HANAKO_TYPE=claude
AGENT_HANAKO_MODEL=claude-sonnet-4-6

# Gemini エージェント "taro"
AGENT_TARO_TYPE=gemini
AGENT_TARO_MODEL=gemini-2.5-flash

# Codex エージェント "jiro"
AGENT_JIRO_TYPE=codex
AGENT_JIRO_MODEL=codex-mini-latest

Discord コマンド一覧

workspace? <name>ワークスペースを切り替え/作成
cost?コストサマリー（累計）
